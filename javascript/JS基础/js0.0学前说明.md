## 0.2学前准备

1： 拒绝二指弹

2：金山打字练习，及格线：100字/分

3：windows 常用快捷键

**CTRL+Y 撤销上一个操作** 

**win+E  打开资源管理器**

**calc运行计算机**

**mapaint运行画图工具**

**notepad：运行记事本**

**ait+f4  退出程序**

**ctrl+alt+del  显示一下选项  锁定计算机，切换应用，注销，更改密码和启动任务管理器**

**ctrl+shift+esc    打开任务任务管理器**
